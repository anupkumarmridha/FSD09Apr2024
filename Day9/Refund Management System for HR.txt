Refund Management System for HR

Objective:
Develop a console-based refund management system to streamline the process of managing refunds for expenses incurred by employees during company-related activities.

Functional Requirements:

1. **Employee Registration:**
   - HR should be able to register employees by providing unique employee IDs, names, and other relevant information.

2. **Expense Submission:**
   - Employees should be able to submit refund requests for eligible expenses incurred during company-related activities.
   - Expenses can include travel expenses, accommodation, meals, etc.
   - Each expense submission should include details such as the expense type, date, description, and amount.

3. **Approval Workflow:**
   - HR should be able to review and approve/refuse expense submissions.
   - Refused expenses should be accompanied by a reason for refusal.

4. **Error Handling:**
   - Ensure appropriate error messages are displayed for invalid submissions or operations.

#### Non-Functional Requirements:

1. **User Interface:**
   - The user interface should be intuitive and user-friendly, providing clear instructions for each operation.

2. **Efficiency:**
   - The system should handle expense submissions and approvals efficiently, minimizing processing times.

3. **Security:**
   - Implement access control measures to ensure that only authorized HR personnel can access and modify employee and expense data.

4. **Scalability:**
   - Design the system to handle a growing number of employees and expense submissions without performance degradation.

#### Optional Enhancements (Not required but could be added for further functionality):

1. **Expense Category Management:**
   - Allow HR to define and manage expense categories for better organization and analysis of expenses.

2. **Email Notifications:**
   - Implement email notifications to notify employees of the status of their expense submissions.

3. **Advanced Reporting:**
   - Enhance expense reporting capabilities by providing advanced filtering and analysis options.
4. **Automatic Approval Rules:**
   - Implement rules-based automatic approval for certain types of expenses below a specified threshold.
